# Contact Book (Python)

A simple Contact Book application built using Python dictionaries. This project allows users to store, search, view, and delete contacts through a menu-driven interface.

## Features

- Add new contacts
- View all contacts
- Search for a contact by name
- Delete a contact
- Prevent duplicate entries with the same name and phone number
- User-friendly menu-driven interface

## Concepts Used

- Python Dictionaries
- Loops (`while`)
- Conditional Statements (`if-elif-else`)
- Dictionary Methods (`items()`, `pop()`)
- User Input Handling

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/contact-book-python.git
   ```

2. Navigate to the project folder:
   ```bash
   cd contact-book-python
   ```

3. Run the program:
   ```bash
   python contact_book.py
   ```
